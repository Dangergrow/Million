# Миллионер — Wealth OS

Современное персональное финансовое приложение с 3D-эффектами, Сейфами целей и динамическим планированием.

![Дашборд](https://github.com/Dangergrow/Million/raw/main/preview.jpg)

## ✨ Возможности

- **Иммерсивный 3D-дашборд** с aurora, tilt-эффектами и частицами
- **Цели + Сейфы** — отдельное хранилище под каждую цель
- **Динамический план** — меняй горизонт, данные сохраняются
- **Категории** (постоянные / временные)
- **Дневные нормы расходов** с контролем
- **Быстрый ввод** доходов и расходов
- **Подробная статистика** за любой период
- Полностью оффлайн (данные в localStorage)
